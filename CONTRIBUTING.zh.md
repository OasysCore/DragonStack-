## 🤝 如何贡献

### 1. 报告问题

发现错误？有建议？

- **GitHub Issues**: [github.com/williamzhao621/dragonstack/issues](https://github.com/williamzhao621/dragonstack/issues)
- **错误报告**: 使用错误报告模板
- **功能请求**: 使用功能请求模板

### 2. 提交 PR

想贡献代码？

1. **Fork** 仓库
2. **创建** 功能分支
3. **做出** 你的修改
4. **测试** 你的修改
5. **提交** PR

### 3. 改进文档

文档就是代码。

- 修复拼写错误
- 添加示例
- 翻译内容
- 澄清说明

### 4. 分享你的经验

- 写博客文章
- 创建视频教程
- 在社交媒体上分享
- 在聚会上演讲

## 📝 贡献指南

### 代码风格

- 遵循现有模式
- 添加中英文注释
- 为新功能编写测试
- 更新文档

### 提交信息

```
type(scope): subject in English

中文描述 / Description in Chinese

Fixes #123
```

类型: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`

### PR 流程

1. **描述** 什么改变了以及为什么
2. **引用** 相关问题
3. **确保** 所有测试通过
4. **请求** 维护者审查

## 🏗️ 开发设置

```bash
# 克隆你的 fork
git clone https://github.com/YOUR_USERNAME/DragonStack-.git
cd DragonStack-

# 安装依赖
npm install

# 运行测试
npm test

# 开始开发
npm run dev
```
