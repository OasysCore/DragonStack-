## 🥇 Trae（推荐）

**出品**: 字节跳动  
**官网**: https://www.trae.ai/

### 优势

- ✅ **完全免费** - 无使用限制
- ✅ **国内网络** - 无需代理
- ✅ **Claude 3.5** - 顶级代码生成能力
- ✅ **Builder 模式** - 多步骤任务执行
- ✅ **中文界面** - 原生中文支持

### 劣势

- ⚠️ **新产品** - 不如 Cursor 成熟
- ⚠️ **插件有限** - 生态较小
- ⚠️ **无 API** - 无法与其他工具集成

### 最适合

- 中国开发者
- 无代理用户
- 注重成本的用户
- 从 Claude Code 切换到国内平台

### 设置

```bash
# 下载地址:
# https://www.trae.ai/

# 安装后:
# 1. 打开 Trae
# 2. 登录（可选）
# 3. 开始 AI 编程
```

## 🥈 Cursor（进阶）

**出品**: Anysphere  
**官网**: https://cursor.sh/

### 优势

- ✅ **功能最强** - 业界领先能力
- ✅ **Composer 模式** - 多文件编辑
- ✅ **丰富插件** - VS Code 兼容
- ✅ **GPT-4 和 Claude** - 多模型支持
- ✅ **终端 AI** - AI 驱动终端

### 劣势

- ❌ **需代理** - 在中国需要 VPN
- ❌ **$20/月** - 付费订阅
- ❌ **免费版有限** - 50 次慢速请求/月
- ⚠️ **访问不稳定** - 中国网络问题

### 最适合

- 有代理用户
- 专业开发者
- 有预算的团队
- 追求最高效率

### 设置

```bash
# 下载地址:
# https://cursor.sh/

# 需要:
# - 稳定代理
# - 信用卡订阅

# 安装后:
# 1. 打开 Cursor
# 2. 登录账户
# 3. 配置代理（如需要）
```
