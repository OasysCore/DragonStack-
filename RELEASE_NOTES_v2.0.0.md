## ✨ What's New

### 🚀 Interactive Onboard
One-command project setup with interactive prompts:
```bash
curl -fsSL https://raw.githubusercontent.com/OasysCore/DragonStack-/main/install.sh | bash
```

Just answer a few questions and your project is ready!

### 🛠️ 6 Complete Skills

| Skill | Command | Purpose |
|-------|---------|---------|
| **brainstorm** | `./dragonstack brainstorm` | Product ideation with 5W1H analysis |
| **design** | `./dragonstack design` | Architecture design documentation |
| **code** | `./dragonstack code` | Code generation guidance |
| **review** | `./dragonstack review` | Code review checklist |
| **test** | `./dragonstack test` | Automated testing |
| **ship** | `./dragonstack ship` | One-click release |

### 💻 Cross-Platform Support
- ✅ macOS
- ✅ Linux  
- ✅ Windows (Git Bash / WSL)

### 🔄 Auto-Update
Already installed? Run the install command again to update:
```bash
curl -fsSL https://raw.githubusercontent.com/OasysCore/DragonStack-/main/install.sh | bash
```

## 📦 Installation

### Quick Start (30 seconds)

```bash
# 1. Install DragonStack
curl -fsSL https://raw.githubusercontent.com/OasysCore/DragonStack-/main/install.sh | bash

# 2. Create a project
~/.dragonstack/scripts/onboard.sh

# 3. Start using
cd your-project
./dragonstack
```

---

## 🎯 Quick Example

```bash
# Start a new project
$ ~/.dragonstack/scripts/onboard.sh
Project name? my-app
Project purpose? A smart todo app
Tech stack? go
AI assistant? trae

# Use DragonStack
$ cd my-app
$ ./dragonstack brainstorm
$ ./dragonstack design
$ ./dragonstack code
$ ./dragonstack review
$ ./dragonstack test
$ ./dragonstack ship
```
